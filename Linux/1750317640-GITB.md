# network_configuration

## Configure Network manager DNS

```bash
nmcli connection show
nmcli connection modify "Wired connection 2" ipv4.ignore-auto-dns yes
nmcli connection modify "Wired connection 2" ipv4.dns "1.1.1.1 8.8.8.8"

nmcli connection modify "Wired connection 2" ipv6.ignore-auto-dns yes
nmcli connection modify "Wired connection 2" ipv6.dns "2606:4700:4700::1111 2001:4860:4860::8888"

nmcli connection down "Wired connection 2" && nmcli connection up "Wired connection 2"
```
